# FarmaNexo Panel

## Descripción

Panel web desarrollado con HTML, CSS, JS y PHP. Permite la visualización y gestión de recetas, usuarios y reportes de manera visual y sencilla. Se integra con la API de FarmaNexo para obtener y actualizar datos.

## Características principales

- Interfaz web simple y modular.
- Integración con la API para mostrar y actualizar información.
- Manejo de usuarios y roles.
- Visualización de estado de recetas en tiempo real.
